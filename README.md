# App Roteiro EUA

App simples em React + Vite para acompanhar a viagem no celular, no estilo do app de treino.

## O que já vem pronto
- Roteiro por dia
- Aba financeira
- Checklist
- Lista de compras
- Anotações
- Progresso da viagem
- Salvamento automático no navegador

## Como rodar no computador
1. Instale o Node.js
2. Abra a pasta do projeto no VS Code
3. No terminal, rode:
   npm install
4. Depois:
   npm run dev
5. Abra o link local que aparecer

## Como publicar no GitHub Pages
1. Crie um novo repositório no GitHub
2. Envie os arquivos deste projeto
3. Depois posso te passar a etapa final do deploy
